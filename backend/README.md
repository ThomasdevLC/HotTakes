
Init back-end : 
`cd backend`   
`npm install`   
`nodemon server`    

Init front-end : 
`cd frontend`   
`npm run start`   

Signup password :
`Minimum length 8`
`Maximum length 20`
`Must have uppercase letters`
`Must have lowercase letters`
`Must have at least 2 digits`
`Should not have spaces`
